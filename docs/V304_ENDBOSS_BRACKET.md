# V3.0.4 Endboss Bracket

- Team strength now uses teamPower(): roster average, team tier, region weight, major/champion aura and tags.
- LCK/LPL have a real endboss advantage; western-only drafts get a penalty.
- Playoff enemy selection is stage-aware and stronger in Semis/Final.
- Bracket matches are decided by teamPower, not pure random.
- Bracket displays each team's path with rating numbers and upset labels.
- Tags now use native browser tooltips only, eliminating visual bleed behind cards.
- Adds Chrono: Robo De Estrella.
