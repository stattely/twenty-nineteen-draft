# Assets Guide

Recommended format:
- PNG transparent or SVG.
- 256x256 for icons.
- 512x512 for team logos.
- Avoid JPG for UI icons because dark mode makes edges look dirty.

Folders:

```txt
public/assets/objectives/
public/assets/ui/
public/assets/chrono/
public/assets/roles/
public/assets/teams-historical/
```
