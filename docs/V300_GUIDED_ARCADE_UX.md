# V3.0.0 Guided Arcade UX

Design changes:
- A single Action Coach tells the player what to do next: Roll, Pick, Sim, Claim.
- Nonessential paragraphs are hidden.
- Meta/Quest/Synergy details stay available but no longer dominate the first screen.
- Player tags are descriptive only on hover.
- Market is visually secondary.
- Selected picks are more prominent.
- Bracket and MVP fixes from V2.9.1 are retained.
