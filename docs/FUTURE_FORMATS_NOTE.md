# Future formats note

Prepared concept directions:
- Worlds: Swiss + knockout (current system)
- MSI: smaller elite field, shorter path, heavier stage pressure
- First Stand: small field, round-robin feel before knockout
- League: regular season + playoffs, league-specific pools (LEC/LCS/LPL/LCK) or mixed

This file is a design note only in V3.0.8 to avoid destabilizing the current playable build.
